# AI 原生笔记本（三栏版）分步开发规划

## 阶段与里程碑

1. 项目初始化与三栏骨架（当前阶段）
2. 左侧树与数据持久化
3. 笔记列表联动与排序
4. 基础编辑器与自动保存
5. AI 模态框与 API Key 设置
6. AI 内容生成与创建新笔记
7. 与笔记对话（上下文拼接）
8. 体验优化与测试

## 目录结构

- `index.html`：入口页
- `src/main.js`：应用入口
- `src/App.vue`：三栏布局与状态流转
- `src/components/*`：左侧、列表、编辑器、AI、设置
- `src/storage.js`：本地存储封装

## 数据模型

- `localStorage/notebookTree`：数组结构表示文件夹与笔记
- `localStorage/userApiKey`：字符串保存 API Key

## 开发步骤详解（当前与后续）

- 初始化项目与骨架：最小可运行的 Vite+Vue，三栏 UI 占位
- 存储模块：提供 `loadData/saveData/loadApiKey/saveApiKey` 与默认示例数据
- 树控件：纯 Vue 实现折叠与选择，后续可替换 Element Plus
- 列表：接收选中笔记本 ID，支持排序
- 编辑器：先用 `textarea`，后续集成 TipTap 并自动保存（防抖）
- AI：模态框 UI 与交互占位，后续接入 OpenAI 与上下文拼接函数

## 验证方式

- 本地启动开发服务器，检查三栏布局与交互占位
- 验证 `localStorage` 读写与默认数据注入
- 后续为关键流程补充轻量单元测试
